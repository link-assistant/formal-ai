# Issue 531 Requirements

The table below converts issue #531 and the initial PR instructions into
checkable requirements. The research and proposal pass has since been followed by
an implementation pass in this same PR: the link-native sequence substrate,
associative deduplication, 1D/2D pattern inference, the seed ontology, and the
solver wiring are now shipped and covered by tests.
