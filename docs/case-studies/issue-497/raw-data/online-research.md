# Online Research

## GitHub UI Traffic Documentation

Source: <https://docs.github.com/en/repositories/viewing-activity-and-data-for-your-repository/viewing-traffic-to-a-repository>

GitHub documents the repository traffic graph under **Insights > Traffic**. The
page says users with push access can view repository traffic including clones,
visitors from the past 14 days, referring sites, and popular content.

## GitHub REST Traffic Documentation

Source: <https://docs.github.com/en/rest/metrics/traffic>

GitHub's REST metrics documentation lists repository traffic endpoints for
clones, page views, popular paths, and popular referrers. The response schemas
expose aggregate counts such as `count`, `uniques`, and per-day traffic entries;
they do not expose individual visitor identities.
