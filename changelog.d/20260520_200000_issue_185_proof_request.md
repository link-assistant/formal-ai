### Added
- Dedicated `proof_request` intent for prompts like "Prove …", "Show that …",
  "Докажи …", "साबित कर …", and "证明 …". The new handler explains the
  formalization pipeline (impulse → formalize → context → plan →
  relative-meta-logic → deformalize → finalize) and, for Gödel-and-determinism
  prompts, asks the user to pick a concrete axiom set so the claim can be
  reduced to a checkable proposition. Closes
  [#185](https://github.com/link-assistant/formal-ai/issues/185).

### Fixed
- Issue [#185](https://github.com/link-assistant/formal-ai/issues/185): the
  prompt "Prove determinism the way logic can handle paradoxes like Godel's
  math incompleteness" no longer returns the generic "I cannot answer that
  from local Links Notation rules yet" fallback. The new handler returns a
  structured response that names the pipeline and the planned
  `relative-meta-logic` integration.
