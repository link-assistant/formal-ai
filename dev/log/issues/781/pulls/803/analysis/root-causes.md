## Why there was no single-line fix

The UI symptom crossed six layers: natural-language intent, symbolic planning,
turn state, protocol output, client MCP routing, and observability. Narration
alone would still leave grep misrouting and Codex namespace failures; namespace
support alone would still produce silent batched turns; logging alone would only
make the bad experience easier to inspect. The complete fix has one shared
planner behavior with thin, tested protocol translations at the edges.
