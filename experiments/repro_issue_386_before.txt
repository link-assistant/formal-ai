TURN 1 (initial list-files program)
PROMPT : Напиши мне программу на Rust, которая выдаёт список файлов в текущей директории
INTENT : write_program
CONF   : 1.00
ANSWER : Вот минимальная программа на языке Rust (list files in the current directory):
         
         ```rust
         use std::fs;
         
         fn main() -> std::io::Result<()> {
             let mut names: Vec<String> = fs::read_dir(".")?
                 .filter_map(Result::ok)
                 .filter(|entry| entry.path().is_file())
                 .map(|entry| entry.file_name().to_string_lossy().into_owned())
                 .collect();
             names.sort();
             for name in names {
                 println!("{name}");
             }
             Ok(())
         }
         ```

INTENT : write_program
CONF   : 1.00
ANSWER : Вот минимальная программа на языке Rust (list files from a path argument in reverse-sorted order):
         
         ```rust
         use std::env;
         use std::fs;

             let path = env::args().nth(1).unwrap_or_else(|| String::from("."));

                 println!("{name}");
         ```
